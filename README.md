🔐 Password Manager (CLI)
A lightweight Python command-line password manager for securely storing and retrieving credentials.
Simple, fast, and perfect for practicing secure storage concepts.

🔧 Features
Add new passwords

Retrieve saved passwords

Optional encryption layer

Local file-based storage

Clean and intuitive CLI interface

🛠 Technologies Used
Python

File handling

Optional encryption

📁 Project Structure
File	Description
main.py	CLI logic
storage.txt	Password storage (encrypted or hashed)


▶️ How to Run
bash
git clone https://github.com/Martun1/Password-Manager-python-CLI-
cd Password-Manager-python-CLI-
python3 main.py
🎯 Purpose
Created to practice secure storage, encryption basics, and command-line application design.
